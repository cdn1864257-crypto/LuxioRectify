⚙️ Comment l’utiliser côté Replit :

Dans ton code backend, tu peux importer ce fichier et sélectionner la langue dynamiquement :

import i18n from './locales/i18n.json' assert { type: "json" };

const lang = user.language || 'fr';
const t = i18n[lang].email_verification;

sendMail({
  to: user.email,
  subject: t.subject,
  html: template
    .replace('[Prénom]', user.firstName)
    .replace('Bonjour', t.greeting)
    .replace('Merci d’avoir rejoint Luxio ✨...', t.message)
    .replace('Confirmer mon compte', t.button)
    .replace('À très bientôt,<br><strong>L’équipe Luxio</strong>', t.footer)
});
